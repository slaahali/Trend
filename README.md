# ترندات تيك توك · السعودية 🇸🇦

منصة ويب بسيطة وتفاعلية تعرض **الهاشتاقات الرائجة على تيك توك** بأسلوب **swarm** (سرب فقاعات متحرّك بفيزياء حقيقية)، إضافةً إلى عرض قائمة قابل للتصفية. البيانات مأخوذة من **Trends MCP**.

> A simple, dependency-free web platform that visualizes trending TikTok hashtags as an animated physics **swarm**, with a filterable list view. Data comes from **Trends MCP**.

---

## ✨ المميزات

- **سرب تفاعلي (Swarm):** كل فقاعة = هاشتاق. حجم الفقاعة = درجة الرواج، ولونها = الفئة. الفقاعات تطفو وتتنافر وتتفاعل مع المؤشر.
- **عرض قائمة (Leaderboard):** ترتيب كامل مع مؤشر درجة رواج لكل هاشتاق.
- **تصفية فورية:** بحث نصّي، تصفية حسب الفئة (18 فئة)، وزر «أبرز ما يهم السعودية».
- **لوحة تفاصيل:** عند النقر على أي هاشتاق تظهر بطاقة فيها الترتيب، درجة الرواج، ورابط مباشر لفتحه على تيك توك.
- **تصميم عربي RTL** بهوية تيك توك (سماوي/ماجنتا على خلفية داكنة)، ويعمل على الجوال.
- **بدون أي تبعيات** — HTML/CSS/JS صِرف، ويعمل حتى بفتح `web/index.html` مباشرة.

## 🔄 تحديث البيانات

البيانات «مخبوزة» داخل `web/data.js` كي تعمل المنصة بدون خادم خلفي. لتحديثها:

1. شغّل أداة Trends MCP:
   `get_top_trends(type="TikTok Trending Hashtags", limit=100)`
2. الصق `as_of_ts` في `RAW.as_of` ومصفوفة `data` في `RAW.rows` داخل
   [`scripts/build-data.mjs`](scripts/build-data.mjs).
3. أعد التوليد:
   ```bash
   npm run build:data
   ```

## ⚠️ ملاحظة عن البيانات (مهمة وصريحة)

قائمة `TikTok Trending Hashtags` في Trends MCP هي قائمة **عالمية** — المصدر **لا يوفّر فلتر دولة**، لذلك هذه ليست قائمة سعودية بحتة. وسم «أبرز ما يهم السعودية» (★) **تحريري للإبراز فقط** (كرة القدم، كأس العالم، البثّ، الموجة الكورية…)، ولا يدّعي ترتيباً سعودياً خاصاً. التصميم يسمح بتركيب أي feed سعودي حقيقي لاحقاً عبر تعديل خطوة التوليد.

كما أن مصدر السلاسل الزمنية لكل هاشتاق على تيك توك (`get_trends`/`get_growth`, source=`tiktok`) لم يُرجِع بيانات لهذه الهاشتاقات، لذلك تُحسب **درجة الرواج** من موضع الهاشتاق في القائمة المرتّبة.
