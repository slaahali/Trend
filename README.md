# ترندات تيك توك · السعودية 🇸🇦

منصة ويب بسيطة وتفاعلية فيها **ثلاثة أقسام**: الهاشتاقات الرائجة على تيك توك بأسلوب **swarm** (سرب فقاعات بفيزياء حقيقية)، **ترندات البحث**، و**بث مباشر رائج (Popular LIVE)**. البيانات مأخوذة من **Trends MCP**.

> A simple, dependency-free web platform with three tabs: trending TikTok hashtags as an animated physics **swarm**, **Trending Searches**, and **Popular LIVE**. Data from **Trends MCP**.

---

## ✨ المميزات

ثلاثة تبويبات في الأعلى:

1. **الهاشتاقات** — سرب تفاعلي (Swarm): كل فقاعة = هاشتاق، حجمها = درجة الرواج، لونها = الفئة، تطفو وتتنافر وتتفاعل مع المؤشر. مع عرض قائمة، بحث، تصفية حسب الفئة (18 فئة)، زر «أبرز ما يهم السعودية»، ولوحة تفاصيل لكل هاشتاق برابط مباشر لتيك توك.
2. **ترندات البحث** — قائمة مرتّبة لأكثر عمليات البحث رواجاً (من Google Trends)، مع تصنيف وفلترة فورية ومؤشر رواج.
3. **بث مباشر رائج (Popular LIVE)** — هاشتاقات البث المباشر الأعلى رواجاً + أكثر فئات البث رواجاً كـ«غرف LIVE».

كل ذلك بتصميم عربي RTL بهوية تيك توك (سماوي/ماجنتا على خلفية داكنة)، يعمل على الجوال، و**بدون أي تبعيات** (يعمل حتى بفتح `web/index.html` مباشرة).

## 🔄 تحديث البيانات

البيانات «مخبوزة» داخل `web/data.js` كي تعمل المنصة بدون خادم خلفي. لتحديثها:

1. شغّل أداة Trends MCP:
   `get_top_trends(type="TikTok Trending Hashtags", limit=100)`
2. الصق `as_of_ts` في `RAW.as_of` ومصفوفة `data` في `RAW.rows` داخل
   [`scripts/build-data.mjs`](scripts/build-data.mjs).
3. أعد التوليد:
   ```bash
   npm run build:data
   ```

## ⚠️ ملاحظة عن البيانات (مهمة وصريحة)

- **الهاشتاقات:** قائمة `TikTok Trending Hashtags` في Trends MCP **عالمية** — المصدر **لا يوفّر فلتر دولة**، فهي ليست سعودية بحتة. وسم «أبرز ما يهم السعودية» (★) **تحريري للإبراز فقط**. درجة الرواج محسوبة من موضع الهاشتاق في القائمة (السلاسل الزمنية لكل هاشتاق على تيك توك غير متوفرة في المصدر).
- **ترندات البحث:** Trends MCP **لا يوفّر** feed لترندات بحث تيك توك، لذا هذا القسم يستخدم **Google Trends** (بيانات بحث حقيقية، عالمية وليست خاصة بتيك توك أو بالسعودية) — والمصدر موضّح داخل الواجهة.
- **بث مباشر رائج:** Trends MCP **لا يوفّر** feed لـ Popular LIVE الفعلي، لذا هذا القسم **مشتق** من إشارات حقيقية: هاشتاقات البث المباشر الرائجة وأكثر الفئات رواجاً (كغرف بث). ترتيب البثوث الفعلي يحتاج TikTok LIVE API.

التصميم يسمح بتركيب أي مصدر حقيقي (سعودي أو خاص بتيك توك) لاحقاً عبر تعديل خطوة التوليد في `scripts/build-data.mjs`.
